# Panvi Chronicles

Panvi Chronicles is a premium travel blog and storytelling platform that showcases travel experiences, hidden destinations, road trips, luxury stays, photography, and adventure journeys across India.

The website was designed with a modern, responsive, and immersive user experience to inspire travelers and document memorable journeys.

## Features

- Modern Responsive Design
- Luxury Travel Blog Layout
- Interactive Destination Showcase
- Travel Story & Journal Section
- Instagram-Style Photo Gallery
- YouTube Integration
- Travel Lessons & Experiences
- Mobile-Friendly Interface
- Smooth Animations & Transitions
- Social Media Integration

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts

## Website Sections

- Home
- About
- Destinations
- Travel Stories
- Gallery
- YouTube
- Travel Lessons
- Footer & Social Links

## Purpose

Panvi Chronicles was created to share authentic travel experiences, inspire exploration, and preserve unforgettable memories through stories, photography, and videos.
